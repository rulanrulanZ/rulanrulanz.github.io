Quick demo of Spatial Aware UI Elements

In a responsive world expanding elements can be a troublesome to manage as a developer. Not much else needs to be said.

For this portfolio I needed to make sure that even if the user is browsing from a small tablet the expanding tiles wouldn't leak outside it's parents bounds so that I could avoid problems with horizontal scrolling.

I extracted the code from my portfolio into a small demo below. Have a play around, fork it, steal it, claim it or give a comment.
